# Sistema de Gestão de Solicitações de Documentos Acadêmicos

API REST para cadastro, consulta e movimentação de solicitações de documentos acadêmicos.

## Tecnologias previstas

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Spring Security
- JWT
- Docker
- OpenAPI/Swagger
